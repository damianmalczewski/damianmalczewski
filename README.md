## 💡 Featured Projects

### Problem4J

A Java library for standardized HTTP API error responses based on [RFC 7807 – Problem Details for HTTP APIs](https://datatracker.ietf.org/doc/html/rfc7807), featuring Spring Boot integrations and custom abstractions for exception-to-response mapping. It extends Spring's built-in `ErrorResponse` and `ProblemDetail` support with a more flexible and expressive programming model.

Visit [project's website](https://problem4j.github.io) for more info.

### iEmu

A simple browser-based emulator for LwM2M IoT devices with support for the **LwM2M Light Control** object (`/3311`).

Visit [project's repository](https://github.com/damianmalczewski/iemu) for more info.
